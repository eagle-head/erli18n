# Support

Thanks for using erli18n! This umbrella ships two Hex packages — the `erli18n`
runtime library and the `rebar3_erli18n` rebar3 plugin. Please use the channel
that matches what you need so questions, bugs, and security reports each land in
the right place.

## Where to go, by intent

- **How-to and usage questions** ("how do I…", configuration help, "is this a
  bug or am I holding it wrong?") — start a
  [GitHub Discussion](https://github.com/eagle-head/erli18n/discussions), or ask
  on the [Erlang Forums](https://erlangforums.com). Please keep the issue tracker
  for confirmed bugs and concrete feature requests.
- **Reproducible bugs and feature requests** — open an issue via the
  [issue templates](https://github.com/eagle-head/erli18n/issues/new/choose).
  Pick the package in the dropdown and include a minimal reproducer — a small
  Common Test case or `.po` fixture makes bugs far faster to fix.
- **Security vulnerabilities** — do **not** open a public issue or discussion.
  Follow the private process in [SECURITY.md](SECURITY.md).

## Supported versions

erli18n targets Erlang/OTP 27, 28, and 29. Only the latest minor of each package
receives fixes; see [SECURITY.md](SECURITY.md#supported-versions) for the exact
support window.

This is a maintainer-led project run on a personal account, so please be patient
with response times.
